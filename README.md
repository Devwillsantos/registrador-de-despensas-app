# Registrador de Despesas

Um aplicativo mobile simples e intuitivo desenvolvido para o gerenciamento de finanças pessoais e controle de gastos diários.
Este projeto foi realizado como parte da disciplina de Desenvolvimento Mobile.

## 🛠️ Tecnologias Utilizadas

* **React Native & Expo**
* **TypeScript**
* **Firebase Auth & Firebase Firestore**
* **Cloudinary API**

## 🚀 Principais Funcionalidades

* Cadastro, leitura, edição e exclusão de transações em tempo real.
* Controle de saldo da carteira com validação de saldo insuficiente.
* Gráficos informativos na tela de estatísticas.
* Funcionamento offline com persistência de dados local via AsyncStorage.
